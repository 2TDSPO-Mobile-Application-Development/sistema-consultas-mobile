# 📱 Sistema de Consultas Mobile

Aplicativo mobile para gerenciamento de **consultas médicas**, permitindo visualizar, confirmar e cancelar consultas de forma simples.

---

## 📌 Sobre

Projeto desenvolvido com **React Native + Expo**, focado em componentização e organização de código.

---

## ⚙️ Funcionalidades

* Visualizar consultas
* Confirmar consultas
* Cancelar consultas
* Exibir informações de médico e paciente

---

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/2TDSPO-Mobile-Application-Development/sistema-consultas-mobile.git
```

2. Acesse a pasta:

```bash
cd sistema-consultas-mobile
```

3. Instale as dependências:

```bash
npm install
```

4. Inicie o projeto:

```bash
npx expo start
```
